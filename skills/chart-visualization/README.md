## 技能说明

### 功能

本技能为 Claude AI 提供数据可视化的专业知识，包括：

1. **图表推荐**：根据数据类型和用户需求推荐合适的图表类型
2. **配置生成**：生成符合 GPT-Vis 语法的图表配置
3. **最佳实践**：提供图表选择和使用的最佳实践指导

### 支持的图表类型

- **统计图表** (17 种)：折线图、柱形图、条形图、饼图、面积图、散点图、双轴图、直方图、箱线图、雷达图、漏斗图、瀑布图、水波图、词云图、小提琴图、韦恩图、矩阵树图
- **关系图表** (6 种)：思维导图、流程图、网络图、桑基图、组织架构图、鱼骨图
- **地图图表** (2 种)：点标注地图、热力地图
- **其他** (2 种)：表格、总结摘要

### 使用场景

当需要以下功能时，使用本技能：

- 数据可视化图表推荐
- 根据数据生成可视化配置
- 选择合适的图表类型
- 理解不同图表的使用场景
- 生成 GPT-Vis 语法的图表代码

## 快速开始

### 1. 基础用法

向 Claude 询问可视化相关的问题，例如：

```
用户：我有销售数据，2020年100万，2021年120万，2022年150万，如何可视化？
```

Claude 将使用本技能分析数据并推荐折线图，生成相应配置。

### 2. 图表推荐

```
用户：我想展示各部门的预算占比，应该用什么图表？
```

Claude 将推荐饼图并说明原因。

### 3. 配置生成

```
用户：帮我生成一个柱形图，比较三个产品的销量：产品A 30件，产品B 50件，产品C 20件
```

Claude 将生成完整的 vis-chart 配置代码。

## 图表选择指南

### 时间序列
- **折线图**：显示趋势变化
- **面积图**：强调总量和趋势

### 分类比较
- **柱形图**：垂直比较，适合短标签
- **条形图**：水平比较，适合长标签

### 占比分析
- **饼图**：部分占整体（≤5个分类）

### 关系展示
- **思维导图**：层级关系
- **流程图**：步骤流程
- **网络图**：复杂网络

### 地理数据
- **点标注地图**：具体位置
- **热力地图**：密度分布
