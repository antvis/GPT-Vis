About GPTVis API

The GPTVis class provides a unified interface for rendering all chart types in GPT-Vis. Instead of creating individual chart instances, you can use a single GPTVis instance and render different chart types by passing configurations with a type field.

Benefits:

  • Single API for all chart types
  • Easy chart type switching
  • Consistent configuration pattern
  • Automatic resource management
