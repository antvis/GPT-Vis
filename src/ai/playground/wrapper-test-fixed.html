About the Wrapper Feature

The wrapper option in GPTVis constructor allows you to display charts with an outer container that includes useful features:

  • Preview and Code tabs for switching between chart view and config view
  • Copy code button to copy the chart configuration
  • Download button to save the chart as an image
  • Zoom in/out buttons for G6 charts (mind-map, network-graph, etc.)

Set wrapper: true in the GPTVis constructor to enable this feature.
