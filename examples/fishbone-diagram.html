Options

Configuration Options

PropertyTypeDescription
datarequired图表的数据，必填，对象类型，表示鱼骨图的根节点（核心问题）；
data.namerequired根节点名称，即核心问题描述，必填，文本类型；
data.childrenrequired原因分类列表（鱼骨主干分支），必填，数组对象类型；
data.children.namerequired分类名称，即原因大类，必填，文本类型；
data.children.childrenoptional具体原因列表（鱼骨细分支），选填，数组对象类型；
data.children.children.namerequired具体原因名称，必填，文本类型；
themestring可选，图表主题风格，可选值: default(浅色)/academy(学术风格)/dark(暗色)，默认: default
titlestring可选，图表标题，显示在图表顶部
style.palettestring[]可选，自定义分支节点颜色列表，十六进制颜色值数组，按分支顺序循环使用，覆盖主题默认色板
style.backgroundColorstring可选，画布背景色，十六进制颜色值，覆盖主题默认背景色
style.texturestring可选，节点渲染风格，可选值: rough(手绘风格)/default(平滑风格)，默认: default
