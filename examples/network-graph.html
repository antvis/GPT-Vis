Network Graph

网络图（Network Graph）是一种展示实体（节点）之间的关系（边）的图。通过节点和边的连接，直观地表示复杂的网络结构。每个节点代表一个实体，而每条边则表示两个节点之间的关系或连接。

Use Cases

  • 社交网络中的人际关系展示，例如朋友关系、关注关系等。
  • 知识图谱中的概念关联，展示实体之间的复杂关系。
  • 分析复杂网络结构中的模式，例如通信网络中的节点连接情况。

Visual Examples

在《哈利波特》系列中，哈利·波特的两个好友是赫敏·格兰杰和罗恩·韦斯莱，伏地魔是哈利的主要敌人并曾试图杀死哈利。用网络图可视化。

用网络图展示公司内部的协作关系：产品经理将需求传递给设计师，设计师输出设计稿给开发者，开发者提测给测试员。

用网络图展示知识图谱中"人工智能"相关概念的关联关系。

Options

Configuration Options

PropertyTypeDescription
typerequired图表的类型，必填，文本类型，值必须为 "network-graph"。
datarequired图表的数据，必填，对象类型，包含 nodes 和 edges 两个字段。
data.nodesrequired网络图中的节点数组，每个节点需包含 name 字段（唯一标识）。
data.edgesrequired网络图中的边数组，每条边需包含 source（起点）、target（终点）、name（关系名称）字段。
layoutoptional布局算法，选填，可选值为 "force" | "circular" | "grid" | "radial" | "concentric" | "dagre"，默认为 "force"。
titleoptional图表的标题，选填，文本类型。
themeoptional图表主题，选填，文本类型，可选值为 "default" | "dark" | "academy"，默认值为 "default"。
