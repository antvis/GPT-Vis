GPTVis Syntax Demo

Render charts using a markdown-like syntax string

About Vis Syntax

The vis syntax provides a concise, markdown-like format for defining chart configurations. This reduces token usage for LLM outputs and provides better error tolerance than JSON.

Syntax Rules:

  • vis [chartType] - Defines the chart type (required)
  • data - Starts a data array section
  • - key value - Array item with key-value pairs
  • key: value or key value - Top-level key-value pair
  • style - Starts a style object section
  • Multiple space-separated values become arrays

Syntax Examples

Pie Chart with Syntax

vis pie data - category 分类一 value 27 - category 分类二 value 25 - category 分类三 value 18 - category 其他 value 30 innerRadius: 0.6

Line Chart with Syntax

vis line data - time 2015年 value 1700 - time 2016年 value 1500 - time 2017年 value 1200 - time 2018年 value 1000 title 出生人口变化 axisXTitle 年份 axisYTitle 出生人口（万人）
